# 📚 Student Attendance Management System

A web-based **Student Attendance Management System** developed using **PHP, MySQL, HTML, CSS, JavaScript, and Bootstrap**. The application helps educational institutions manage student attendance efficiently through an easy-to-use interface.

---

## 🚀 Features

* 👨‍🎓 Student Registration
* 📋 Manage Student Records
* ✅ Mark Daily Attendance
* 📅 View Attendance History
* 🔍 Search Student Records
* 📊 Attendance Reports
* 💾 MySQL Database Integration
* 📱 Responsive User Interface

---

## 🛠️ Technologies Used

* PHP
* MySQL
* HTML5

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/yuktapatil18/student-attendance-management-system.git
```

### 2. Move the project

Copy the project folder to

```
C:\xampp\htdocs\
```

### 3. Start XAMPP

Start:

* Apache
* MySQL

### 4. Create Database

Open

```
http://localhost/phpmyadmin
```

Create a database and import the SQL file located in

```
database/dbattendance.sql
```

### 5. Run the Project

Open your browser

```
http://localhost/attendance
```
